☁️ 동기화 설정

Firebase를 연결하면 모든 기기에서 할 일이 실시간으로 동기화됩니다.
최초 1회만 설정하면 이후 자동 연결됩니다.

  • 1

    console.firebase.google.com 접속 → 구글 로그인 → 프로젝트 추가

  • 2

    프로젝트 홈 → 웹 앱 추가(</>) → 앱 등록 → firebaseConfig 복사

  • 3

    왼쪽 메뉴 → Firestore Database → 데이터베이스 만들기 → 테스트 모드 선택

  • 4

    아래 입력 후 연결. 다른 기기도 동일하게 입력하면 자동으로 동기화됩니다.
